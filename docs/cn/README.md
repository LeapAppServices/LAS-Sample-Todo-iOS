# LAS-Sample-Todo-iOS

## 介绍

这个项目介绍如何使用 LAS 展示一个列表，并向其中添加或者删除元素。

## 演示功能

- 创建 LASObject
- 查询 LASObject
- 删除 LASObject

## 如何运行

- 克隆这个仓库，然后打开项目
- 在 leap.as 控制台中创建一个应用，下面称他为 LAS 应用。如果已经创建，跳过这个步骤。
- 在 `AppDelegate.` 中填写 LAS 应用的 applicationId 和 clientKey.
- 按下 Commond + R 按钮运行
